Descripción

El objetivo de este curso es enseñaros la semántica correcta de HTML5 actualizada al último estándar oficial.

En el curso seguimos paso a paso la documentación oficial para que no sea una interpretación, si no una explicación de la documentación oficial.

Espero que este curso os ayude a iniciaros en el desarrollo web de forma correcta.


Lo que aprenderás

  • En este cursos aprenderás el lenguaje base para realizar paginas web

¿Hay requisitos para realizar el curso?

  • El curso empieza desde 0, no necesitas ningún conocimiento previo.

¿Para quién es este curso?

  • Cualquier persona que quiera iniciarse en el desarrollo web.
